# 🚀 Астероидный рудокоп (Asteroid Miner)

[![Python](https://img.shields.io/badge/Python-3.7%2B-blue.svg)](https://www.python.org/)
[![Pygame](https://img.shields.io/badge/Pygame-2.0%2B-green.svg)](https://www.pygame.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Космическая аркада на Python/Pygame, где вы управляете кораблем-шахтером, добываете ценные минералы с астероидов и сражаетесь с космическими пиратами!


## 📋 Описание

В далеком будущем человечество истощило ресурсы Земли. Вы - пилот космического рудокопа, отправляющийся в пояс астероидов для добычи ценных минералов. Но будьте осторожны! Космические пираты не дремлют и готовы уничтожить любого, кто посягнет на их территорию.

## 🎮 Геймплей

### Основные механики:
- ⛏️ **Добыча минералов** - подлетите к астероиду и нажмите SPACE для добычи
- 🛡️ **Выживание** - уклоняйтесь от вражеских кораблей и следите за здоровьем
- 🌊 **Система волн** - каждая новая волна приносит больше врагов и астероидов
- 💎 **Экономика** - собирайте минералы для получения очков
- ❤️ **Система жизней** - у вас есть 3 жизни, используйте их с умом

### Типы астероидов:
| Тип | Размер | Ценность | Прочность |
|-----|--------|----------|-----------|
| 🟤 Малый | 10-20 px | 1-3 минерала | 1 удар |
| 🟠 Средний | 20-35 px | 3-7 минералов | 2 удара |
| 🔴 Крупный | 35-50 px | 7-15 минералов | 3 удара |

### Враги:
- 🔴 **Космические пираты** - преследуют игрока и наносят урон при столкновении
- Скорость и частота появления врагов увеличивается с каждой волной

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| `W` / `↑` | Движение вперед |
| `S` / `↓` | Движение назад |
| `A` / `←` | Поворот влево |
| `D` / `→` | Поворот вправо |
| `SPACE` | Добыча минералов |
| `P` | Пауза |
| `R` | Перезапуск игры |
| `ESC` | Выход из игры |
